# WeChatGroupDaily

「🦞 奇奇怪怪养龙虾群」的群日报归档。

## 这是什么

这个文件夹存放的是微信群「🦞奇奇怪怪养龙虾群-自迭代无限生」的定期日报，由 AI 自动生成，涵盖群内的讨论精华、知识沉淀和重要信息。

每期日报以**时间段**为单位命名，从 2026-05-07 开始持续归档。每天下午 14:00 自动生成当天日报。

## 文件格式

每期日报同时输出两个版本：

| 格式 | 说明 |
|------|------|
| `🦞奇奇怪怪养龙虾/html/*.html` | 带排版样式的网页版，用浏览器打开 |
| `🦞奇奇怪怪养龙虾/png/*.png` | 截图版，方便转发和分享 |

## 目录结构

```
.
├── README.md
└── 🦞奇奇怪怪养龙虾
    ├── html    # 网页版日报
    └── png     # 截图版日报
```

## 命名规则

```
🦞奇奇怪怪养龙虾群<日期或日期范围>.<格式>
```

- 单日：`🦞奇奇怪怪养龙虾群2026-05-23.png`
- 跨天：`🦞奇奇怪怪养龙虾群2026-05-07 → 2026-05-13.html`
