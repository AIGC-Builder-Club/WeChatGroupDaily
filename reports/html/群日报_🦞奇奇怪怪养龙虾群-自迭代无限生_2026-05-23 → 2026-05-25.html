Grill → Spec → Goal 三阶工作流
Yanbo. / 小严同学 · 20:18
  1. 用 SuperPowers 或 grill me 进行需求澄清，出完整 spec
  2. 把 spec 作为 goal 的目标描述——所有边界和完成条件前置确定
  3. 在 goal 模板中引用 templates/README.md 作为指导文件
  4. Goal 执行中保存所有中间产物，方便复盘时修正路径
  5. 对 grill 本身也可 grill，对 goal 本身也可 grill——元操作递归
问题前置确定 + 中间产物可复盘 + 第二次基本不会错
